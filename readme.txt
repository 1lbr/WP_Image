New image API's for WordPress

== Description ==

At this moment only GD is supported. With this code there will also be ImageMagick support.
Also the current code is partly rewritten to make it easier to generate thumbnials for certain images

== Installation ==

1. Upload the folder `tabify-edit-screen` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings -> WP_Image to enable tabs for a specific post type
